## ⚡ O que é o try-catch e como ele se conecta com a Prevenção de Erros?
O try-catch é uma estrutura usada para tratar erros de forma controlada, onde o o bloco try contém o código que pode gerar uma exceção enquanto o catch captura esse erro e define como lidar com ele evitando que o programa pare inesperadamente. Embora não impeça que erros aconteçam, ele é essencial na prevenção de falhas maiores, pois permite tratar problemas, exibir mensagens adequadas e manter o sistema funcionando mesmo diante de situações inesperadas.
